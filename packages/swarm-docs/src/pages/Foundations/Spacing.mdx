# Spacing

<br/>
Swarm uses both a 4-base and 5-base grid scale depending on viewport size. This relative size relationship helps maintain balance and visual harmony for the right screen. It also allows for engineers to produce less viewport specific code.

**Application**

Swarm spacing is built into all atoms of the design system, like Buttons and Text Inputs. To assemble layouts, spacing tokens can be applied to all four edges of a DOM element, using margin and padding properties.
