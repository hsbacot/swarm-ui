## Sizing

** Size & Height **

Any button can be configured to one of three distinct sizes. Size dictates label style and top/bottom padding. Button padding uses spacing variables which are dynamic based on viewport.
<br/>
<img src="/docs-images/button-sizing-size.png" width="600" height="373" />
<br/><br/>

** Width **

Any button can be configured to one of three width settings. Width dictates left/right padding behavior. Button padding uses spacing variables which are dynamic based on viewport.
<br/>
<img src="/docs-images/button-sizing-width.png" width="600" height="371" />
<br/><br/>

## a11y
Keyboard interactions: The button action should be triggered by the enter and space keys.
